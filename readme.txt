Touch enabled jQuery plugin that lets you create beautiful responsive carousel slider, Just install and build slider in a few minutes, It allows you to easily create logo carousel/slider to display logos of clients, partners, sponsors, affiliates etc.

== Description ==
Fully written in jQuery, touch enabled wordpress plugin based on OWL Carousel that lets you create beautiful responsive carousel slider, it works very fast. You can add images from computer, and upload a bunch of new images from Media Library at once.


<strong>Seo Carousel Slider Features: </strong>

- Image Slide (New)
- Text Slide (New)
- Simple, easy to use interface
- Full width slideshow support
- Fully Customisable
- Multiple carousel on single page
- Unlimited Slides
- Touch and Drag Support
- Fully Responsive
- Modern Browsers
- Infinity Loop
- Ability to make almost all options responsive
- Uses minified CSS, JavaScripts and jQuery files. Lazy loading keeps the initial amount small
- Working in Chrome, Firefox, Safari, IE7-11, Opera
- SEO friendly
- FontAwsome & Icon moon supported

== Installation Guide ==

**Installation steps**
1.Download the package.
2.Extract the contents of plugin.zip to wp-content/plugins/ folder
3.Activate the Plugin in WP-Admin.
4.Goto Media Slider.
